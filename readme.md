## Termux
```bash
$ pkg upgrade && pkg update
$ pkg install git -y
$ pkg install nodejs -y
$ pkg install ffmpeg -y
$ pkg install imagemagick -y
$ git clone https://github.com/oziispedzz/FureyaMd
$ cd FureyaMd
$ npm i
$ npm start 
```
#### For android 10 and above don't use npm, use yarn install
```bash
$ pkg install yarn -y
$ yarn install
```
